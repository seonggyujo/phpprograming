# 이미지 게시판

## 개발 언어
PHP, MySQL, HTML, CSS, JavaScript

## 프로젝트 설명
이미지 업로드가 가능한 게시판 시스템
## 구현 기능
- 게시글 CRUD
- 이미지 업로드
- 댓글 기능
- 검색
- 페이지네이션

## 파일 구조
- board_list.php - 목록
- board_write.php - 작성
- board_view.php - 상세보기
- board_edit.php - 수정
- board_delete.php - 삭제
- comment_write.php - 댓글 작성
- comment_delete.php - 댓글 삭제
